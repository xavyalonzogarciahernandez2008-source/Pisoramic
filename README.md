# Distribuidora Pisoramic - Sistema de Inventario Web

Una aplicación web moderna para gestionar el inventario de productos de la Distribuidora Pisoramic.

## Características

- ✅ Agregar productos con imágenes
- ✅ Ver lista completa de productos
- ✅ Editar productos y cambiar imágenes
- ✅ Eliminar productos
- ✅ Búsqueda y filtrado en tiempo real
- ✅ Visualización de valores totales
- ✅ Alertas de bajo stock

## Requisitos

- Node.js (v14 o superior)
- npm

## Instalación

1. Instala las dependencias:
```bash
npm install
```

## Desarrollo

Para ejecutar en modo desarrollo:
```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`

## Producción

Para ejecutar en producción:
```bash
npm start
```

## Estructura del Proyecto

```
inventario/
├── public/           # Archivos estáticos (HTML, CSS, JS)
├── src/              # Código del servidor
├── data/             # Almacenamiento de datos
└── package.json      # Configuración de dependencias
```
